搜索引擎爬虫规则

爬虫的定义

爬虫是搜索引擎的一个程序，用途是为搜索引擎发现一个网页。主要功能有：爬取新页面、回访老页面、智能发现和回访。因此如果爬虫来的多，收录就会快速，也说明该网站是个优质网站。

如何查看爬虫次数

可以在相应的搜索引擎资源平台查看，如今一般都是用百度搜索的，因此可以进入百度资源搜索平台查看爬虫的爬取频次
百度资源搜索平台查看爬虫爬取频次需要进行登陆操作，并且绑定网站

导致爬虫不友好的情况

  • 中文网址
  • 打开速度慢（最好控制在500-1000ms之内）
  • 站内有死链

提升爬虫友好度的方法

  • 优质的网站内容
  • 每日的内容更新量
  • 定时的更新内容（有规律的更新，可以让爬虫也有规律的爬取）
  • 优质的网站内链
    网站要养爬虫三个月

搜索引擎友好度规则

网页的识别

  • 文字
    • 丰富度（文字内容是否足够多）
    • 专业度
  • 图片
    • 清晰度
    • 原创性（图片和其他图片比较是否相似（如百度识图是否能够识别））
    • 相关性（图片内容是否和文字及网站相关）

提升网站友好度

  • 网站内容能被识别
  • 网站打开速度快
  • 扁平化的网站结构
  • 网站无死链接
  • 网站无违法内容

网站TDK书写

知识要点

  • 标题写作原则

网站三要素TDK

  • TDK是什么
    • Title 标题
    • Description 描述
    • Keywords 关键词
  • 关键词与标题
    • 分词
      标题要写入（包含）要做的词、用户搜索的词
      1. 把标题分成关键词、短语
      2. 把用户搜索的关键词分出来
      3. 把我们要做的关键词分出来
    • 匹配
      标题中设置的关键词尽量匹配用户搜索词
      1. 完全匹配
      2. 部分匹配
      3. 不匹配

定位关键词

  • 挖掘关键词并筛选分析
  • 核心关键词
    • 确定页面核心 单个核心的标题更加利于排名
    • 这个页面是什么？做什么？
  • 需求关键词
  • 注意
    • 避免关键词太宽泛、不精准
    • 注意一词多义/同义词/近义词（优先选择指数高的/长尾词数量多的词）

网站标题书写

TDK示例：
Title：小风扇-小风扇厂家-小风扇批发-小风扇价格-旋风小风扇
Keywords：小风扇，usb小风扇，小风扇厂家，小风扇批发，小风扇价格
Description： 旋风小风扇厂家生产各类小风扇，包括USB小风扇，迷你小风扇等。小风扇批发价格实惠，欢迎前来定制小风扇。

  • 组合关键词写成标题
    • 关键词1+关键词2+关键词3>品牌词 关键词=核心词+需求词
    • 根据页面定位选择关键词
      • 匹配 1、该页面核心定位
      • 指数 2、需求量 3、流量
      • 长尾词数量 相关需求量大小
      • 转化率
        建议3-5个关键词左右
  • 首页标题
    • 关键词选择
      首页为整站的核心，因此应当选择能够包含整个网站的关键词
    • 注意
      网站一般不建议写产品名称，产品名称可以写在产品栏目分类中
    • 首页栏目页的描述和Keywords
      • 关键词
        1. 对排名不起作用
        2. 就写这个页面标题包含的关键词
        3. 可使用站长工具查询关键词排名，使用英文逗号隔开
      • 描述
        1. 字数60-80字
        2. 具备吸引性
        3. 把标题的关键词加进去，写成通顺的几个句子
        4. 写描述是什么作甚，并以此拓展
  • 栏目页标题
    • 确定栏目核心 1、分类关键词 2、栏目分类
    • 挖掘分析需求，选择关键词 1、以栏目核心进行拓展 2、选择跟栏目一致的关键词
    • 整合关键词写好标题 栏目核心关键词+需求词=品牌词
  • 内页（文章页）TDK
    • 文章页标题
      1. 文章页面标题=文章标题+品牌词
      2. 标题包含长尾关键词
      3. 吸引点击的标题
    • 描述
      一般自动截取文章第一段
    • 关键词
      长尾关键词

网站标题优化

优秀的标题

  • 可以组成一段通顺的话
  • 吸引用户的点击
  • 尽可能匹配到更多的关键词

时间性

信息量大

信用度

以稀缺需求展开

写标题注意事项

关键词

  • 关键词和keywords的区别
  • 建议一个标题3个关键词左右+品牌关键词
  • 靠前的关键词权重越大

关键词堆砌

  • 不要多次出现重复的词
  • 标题是独一无二的

标题长度

  • 标题过长/关键词过多会导致超出部分用户看不到，
  • 大于4个小于30个汉字
  • 不要有过多的描述

符号

一般建议使用英文符号，且不影响阅读。如(_/-/,)

评价词

企业网站不要出现第三方评价词

广告

不要违反广告法（极限词、绝对化用语）

标题修改注意

  • 承受降权的风险 1、排名、权重下降
  • 不破坏原有的匹配 1、原有的关键词，新标题中要包含
  • 想好再去修改 1、要制定好方案 2、要一次性修改完毕

站外设置

网站大量页面组成集合，页面主体部分由内容组成

流程：
上线/更新-蜘蛛抓取-过滤处理-收录保存

有效收录：

1、做用户搜索词的长尾词
2、标题优化
3、换友情链接
4、满足用户刚需

1、做用户会搜索的话题，在用户内容中要满足
2、换友情链接
3、满足用户查询刚需，比如质量下载需求
4、暂时不需要改标题

无效收录：

文章不收录
收录很多，但无排名
原创内容也无排名

页面质量评分计算：
1、内容长度：采集 < 随意聚合 < 空页面 < 一段话或者一句话 < 整篇概括清晰 < 超丰富度内容
2、内容排版
3、需求覆盖
4、页面可读
