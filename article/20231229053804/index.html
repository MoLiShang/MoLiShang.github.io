Zotereo使用

Hokori2023-12-292024-10-31

插件列表

本文所有插件均基于Zotero6.30，如果在安装插件时出现如下错误，则是插件版本无法与Zotero版本兼容，请自行找到兼容版本

Jasmium

Jasmium:一个简单的 Zotero 中文文献管理插件
  1. 拆分或合并 Zotero 中条目作者姓和名
  2. 根据知网上下载的文献文件来抓取引用信息（就是根据文件名）
  3. 添加中文PDF/CAJ时，自动拉取知网数据，该功能默认关闭。需要到设置中开启，注意添加的文件名需要含有中文，全英文没有效果（还是根据文件名）
  4. 为知网的学位论文 PDF 添加书签
  5. 更新中文 translators
  6. 拉取文献引用次数，是否核心期刊

Jasmium github地址：https://github.com/l0o0/jasminum/releases

Better Bibtex for Zotero

Better Bibtex for Zotero: 将文献导出成bibtex文件，和texStudio结合很好用

将文献批量按照自己指定的格式导出到bibtex，形成参考文献。

Better Bibtex for Zotero：https://github.com/retorquere/zotero-better-bibtex/releases

DOI Manage

DOI Manage: 将文献导出成bibtex文件，和texStudio结合很好用

管理文献的DOI，以助于更好的获取文献（有时候长的DOI没办法获取文献PDF，换成短的DOI则可以）

DOI Manage：https://github.com/bwiernik/zotero-shortdoi/releases

Duplicates Merge for Zotero

Duplicates Merge for Zotero: 批量检测文献中的重复文件，并删除重复文件

批量检测文献中的重复文件，并删除重复文件

Duplicates Merge for Zotero：https://github.com/frangoud/ZoteroDuplicatesMerger/releases

Green Frog

Green Frog: 更新文献期刊信息

插件安装后在分类及条目上右击会出现从easyScholar更新期刊信息，点击将根据条目语言从easyScholar获取期刊或会议论文（需要将会议名称填入Conference Name字段）JCR分区、中科院分区基础版、中科院分区升级版、影响因子和5年影响因子、EI及中文期刊更新是否南南农大核心期刊、南农大高质量期刊（仅含食品科学与工程）、中国科技核心期刊、CSCD收录、北大/南大核心、科技核心、EI，SSCI、AJG、UTD24、FT50、CCF、FMS、JCI、AHCI、复合影响因子，综合影响因子、各个大学对期刊的分类及easyScholar自定义数据集（详见easyScholar，如果要使用自定义数据集，需要在easyScholar网站中登录，添加并在easyScholar浏览器插件中选中需要的自定义数据集）并保存在Extra字段，如果显示不正常请先清除其它（Extra）字段。并可在Edit-Preferences-Green Frog中设置哪些字段在列中显示（默认全部显示，如果不需要可以关闭），然后在列上右击即可显示相应字段

Green Frog：https://github.com/redleafnew/zotero-updateifsE/releases

Sci-Hub Plugin for Zotero

Sci-Hub Plugin for Zotero: 通过Sci-hub获取文献

通过Sci-hub获取文献

Sci-Hub Plugin for Zotero：https://github.com/ethanwillis/zotero-scihub/releases

Zotero Reference

Zotero Reference: 一键生成参考文献条目

Zotero-reference 是一款强大的引用管理插件，它可以帮助用户收集、组织、引用和共享文献资源。它可以自动检测文章中的引用，并将其保存到用户的 Zotero 库中，以便以后使用。此外，它还可以自动生成参考文献，并将其导出到文档中，以便用户可以快速完成文献研究。

Zotero Reference：https://github.com/MuiseDestiny/zotero-reference/releases

Zotero Better Notes

Zotero Better Notes: 笔记插件

Zotero Better Notes可以设置主笔记以大纲形式管理其他文献笔记，并结合Zotcard可以写的好看。

Zotero Better Notes：https://github.com/windingwind/zotero-better-notes/releases

Zotcard

Zotcard: 笔记卡片插件

它提供了卡片模板（如概念卡片、角色卡片、金句卡片等，默认可以自定义其他卡片模板），因此可以快速写卡片。此外，它还可以帮助您对卡片进行排序并标准化卡片格式。

Zotcard：https://github.com/018/zotcard/releases

Zotcard模板

检索引擎管理

在Zotero软件界面，从“Zotero首选项 -> 高级 -> 文件和文件夹”打开数据库文件夹，再打开locate/engines.json文件，该文件就是控制检索引擎的文件了，我们只需要修改该文件的内容就可以自定义检索引擎了！
